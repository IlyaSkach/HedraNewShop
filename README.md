# HedraNewShop

Интернет-магазин напитков на React.

## Запуск

```bash
npm install
npm start
```

## Сборка

```bash
npm run build
```

## Структура

```
src/
├── components/    - React компоненты
├── data/          - Данные (товары, контакты)
├── hooks/         - useCart hook
└── styles/        - CSS файлы
```

## Настройка

### Контакты

`src/data/constants.js` - телефон, WhatsApp, Telegram, VK

### Товары

`src/data/products.js` - цены, названия, бренды

### Стили

`src/index.css` - цвета, шрифты
